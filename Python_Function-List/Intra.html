How to solve this exercice?




The subject asks you to write a function, so you should only submit a
 file with the function inside. The Sentinel will import your function 
in its test file.
To test your function, it's as easy as calling 
it after you have defined it! (Look at the video in the e-learning, it 
walks you through an example.) Of course, don't forget to remove your 
tests from your submission file.
Finally, read carefully the subject. Does the function return something...?
